# 깃허브 잔디를 심어보자(Win10)

> 부팅할 때마다 깃허브 잔디를 1번씩 심어줍니다. 

<br>

## 환경

- Windows 10
- Git bash

<br>

## 실행 과정



1. git clone https://github.com/HolaJun/LetsGrowUpGrass.git

   (또는 ZIP download)

2. cd LetsGrowUpgrass

3. git init

4. LetsGrowUpGrass.bat

<br>

## 시작프로그램 등록

1. Win+R → shell:startup
2. LetsGrowUpGrass.lnk(바로가기 파일) 복사 후 붙여넣기
